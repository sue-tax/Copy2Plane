# Copy2Plane

クリップボード内の文字列から、カンマや改行などを削除します。

Excelから、達人シリーズに数字をコピーする際に使用することを想定しています。

- copy2plane Ctrl+Shift+Vで加工する、終了はタスクマネージャーから操作
- copy2planeImm 即座に自動加工する。終了は空のセル2つをコピー
- copy2planeImmTray 即座に自動加工する。終了はタスクトレイからか、空のセル2つをコピー
- copy2planeET タスクトレイから加工を選択する。文字変換機能あり。終了はタスクトレイから。

## copy2plane

### 使用方法

copy2plane.exe を起動します。

文字列をCtrl+Cなどでクリップボードにコピーした後で、Ctrl+Shift+Vを押すと、クリップボード内の文字列下記の加工を行います。

- カンマ、タブ、改行、半角空白、円マーク（\）の削除
- △、▲をマイナス（-）に置換
- マイナス（-）だけの文字列をゼロ（0）に置換

~~終了するときは、Ctrl+Shift+Qを押します。~~ 機能しないようです

## copy2planeImm

### 使用方法

copy2planeImm.exe を起動します。

文字列をCtrl+Cなどでクリップボードにコピーすると、即座に、クリップボード内の文字列に下記の加工を行います。

- カンマ、タブ、改行、半角空白、円マーク（\）の削除
- △、▲をマイナス（-）に置換
- マイナス（-）だけの文字列をゼロ（0）に置換

加工前の文字列が空の文字列でなく、加工後の文字列が空の文字列だと終了しますので、終了させたいときは空のセルを２つ同時にコピーしてください。

## copy2planeImmTray

copy2plane.jpg ファイルを、copy2planeImmTray.exe ファイルと同じフォルダに保存してください。

加工したとき、終了するときに、メッセージを表示します。

### 使用方法

copy2planeImmTray.exe を起動します。

文字列をCtrl+Cなどでクリップボードにコピーすると、即座に、クリップボード内の文字列に下記の加工を行います。

- カンマ、タブ、改行、半角空白、円マーク（\）の削除
- △、▲をマイナス（-）に置換
- マイナス（-）だけの文字列をゼロ（0）に置換

加工前の文字列が空の文字列でなく、加工後の文字列が空の文字列だと終了しますので、終了させたいときは空のセルを２つ同時にコピーしてください。

タスクトレイ内のアイコンを右クリックして、「終了」を選択でも終了します。

## copy2planeET

copy2plane.jpg ファイルを、copy2planeET.exe ファイルと同じフォルダに保存してください。

加工したとき、変換したときに、メッセージを表示します。

### 使用方法

copy2planeET.exe を起動します。

文字列をCtrl+Cなどでクリップボードにコピーします。

タスクトレイ内のアイコンを右クリックして、「加工」を選択すると、クリップボード内の文字列に下記の加工を行います。

- カンマ、タブ、改行、半角空白、円マーク（\）の削除
- △、▲をマイナス（-）に置換
- マイナス（-）だけの文字列をゼロ（0）に置換

タスクトレイ内のアイコンを右クリックして、「大文字→小文字」などを選択すると、クリップボード内の文字列の変換を行います。変換内容と変換例（変換前は`123abcXYZ１２３ｄｅｆＳＴＵあいうカキクｻｼｽ`）は、下記です。

- 大文字→小文字	半角・全角の大文字のアルファベットを小文字に変換します
   `123abcxyz１２３ｄｅｆｓｔｕあいうカキクｻｼｽ`

- 小文字→大文字	半角・全角の小文字のアルファベットを大文字に変換します
  `123ABCXYZ１２３ＤＥＦＳＴＵあいうカキクｻｼｽ`

- 英数字全角→半角	全角のアルファベット・数字を半角に変換します
  `123abcXYZ123defSTUあいうカキクｻｼｽ`

- 英数字半角→全角	半角のアルファベット・数字を全角に変換します
  `１２３ａｂｃＸＹＺ１２３ｄｅｆＳＴＵあいうカキクｻｼｽ`

- ひらがな→カタカナ	全角のひらがなを全角のカタカナに変換します
  `123abcXYZ１２３ｄｅｆＳＴＵアイウカキクｻｼｽ`

- カタカナ→ひらがな	全角のカタカナを全角のひらがなに変換します
  `123abcXYZ１２３ｄｅｆＳＴＵあいうかきくｻｼｽ`

- カタカナひらがな→半角	全角のカタカナ・ひらがなを半角のカタカナに変換します
  `123abcXYZ１２３ｄｅｆＳＴＵｱｲｳｶｷｸｻｼｽ`

- カタカナ→全角	半角のカタカナを全角のカタカナに変換します
  `123abcXYZ１２３ｄｅｆＳＴＵあいうカキクサシス`

タスクトレイ内のアイコンを右クリックして、「終了」を選択でも終了します。
