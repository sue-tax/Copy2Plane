# Copy2Plane

クリップボード内の文字列から、カンマと改行を削除します。

Excelから、達人シリーズにコピーする際に使用することを想定しています。

## copy2plane

### 使用方法

copy2plane.exe を起動します。

文字列をCtrl+Cなどでクリップボードにコピーした後で、Ctrl+Shift+Vを押すと、クリップボード内の文字列から、カンマとタブと改行を削除します。

~~終了するときは、Ctrl+Shift+Qを押します。~~ 機能しないようだ

## copy2planeImm

copy2planeImm.exe を起動します。

文字列をCtrl+Cなどでクリップボードにコピーすると、即座に、クリップボード内の文字列から、カンマとタブと改行を削除します。

削除後の文字列が空の文字列だと終了しますので、終了させたいときは空のセルをコピーしてください。
